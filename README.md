** Private Chef Nurture Funnel **
Overview
An n8n workflow designed to automate the lead intake and qualification process for a private chef service.

Functional Logic
Trigger: Webhook/Form submission from prospective clients.

Processing: Parses inbound data (guest count, date, dietary requirements).

Nurture: Automates follow-up communication and sends menu documentation.

Action: Syncs lead data to Google Sheets and triggers a notification for the chef.

Tech Stack
Platform: n8n

Format: JSON

Integrations: Gmail, Google Sheets
